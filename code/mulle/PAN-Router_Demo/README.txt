Mulle PAN-U Demo

Demo software for the Mulle Platform.

Mulle_PAN-U_demo.c describes normal setup of the Bluetooth PAN-U stack 
and user functions to react on BT events.

To run the program, download it to the Mulle platform according to the
instructions in the Mulle Expansion Board User Manual.
Press the push-button on the Expansion board (INT4) to initiate a Bluetooth connection
towards a mobile phone or camputer.

Set TARGET_BDADDR to the MAC address of the receiving unit.
